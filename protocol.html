NANDA Protocol

An open protocol extending MCP for a decentralized network of AI agents, resources, and tools. The NANDA Protocol provides a comprehensive framework for enabling decentralized AI agent networks to discover, authenticate, transact, and build reputation with each other. Built as an extension to the Model Context Protocol (MCP), NANDA adds the infrastructure needed for large-scale agent ecosystems.

Version 1.0 Alpha

Discovery

Mechanisms for AI agents to find each other in the network, regardless of scale or topology

Commerce

Seamless transactional framework for agents to exchange services and resources

Authentication

Robust verification protocols ensuring secure and trustworthy agent interactions

Reputation

Verifiable agent-to-agent accountability through transparent reputation management
