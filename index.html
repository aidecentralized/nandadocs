Built on Model Context Protocol

NANDA extends the Model Context Protocol (MCP) from Anthropic, adding capabilities for large-scale decentralized agent networks.

Designed for compatibility with MCP, NANDA lets existing MCP-compliant tools join this broader agent ecosystem with minimal changes. Developers can connect their AI applications or services via lightweight NANDA servers and immediately participate in a growing agent network.

As the agentic AI landscape expands toward millions of distributed agents, resources, and tools, NANDA provides the infrastructure for discovery, commerce, authentication, and reputation at scale—all while maintaining sovereignty and privacy.

What is NANDA?

Transforming isolated AI agents into a collaborative "Internet of AI Agents"

NANDA (Networked AI Agents and Decentralized Architecture) is an open protocol and framework that transforms isolated AI agents into a collaborative network. By extending Anthropic's Model Context Protocol (MCP) to multi-agent systems, NANDA enables AI agents, services, and tools to discover each other, authenticate identities, transact value, and build trust – all without centralized control.

Whether you are building a new AI agent from scratch or connecting an existing LLM app to a larger network, NANDA provides the protocols and infrastructure to unlock collaborative intelligence. Continue to the Introduction to understand why NANDA is needed and what it makes possible.
